TERMS

Terms for the private profile lookup.

Public profile data only

This tool is designed around data currently exposed on public TikTok profile pages. If TikTok does not expose a field, the tool should not invent or infer it.

Private use

This version is a private internal build. It does not include storefront, payment, or public sales content.

Platform changes

The live parser depends on TikTok's public page structure, which may change over time. If the site markup changes, the server parser may need an update.

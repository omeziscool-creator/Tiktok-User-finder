PRIVATE TIKTOK LOOKUP

Search a public account, favorite it, and open it again in one click.

This private build fetches public TikTok profile data through the local Node server, shows the account details in the UI, and stores favorites in your browser so you can reopen tracked accounts quickly.

@

A plain HTML file cannot reliably fetch live TikTok data by itself because of browser cross-origin limits. If this page is opened from file:// or another local port, it will automatically call http://127.0.0.1:3000 for live lookups.

TOOLS

All requested features are represented in the interface.

User Lookup

Loads a public profile by handle and fills the main result card with the latest live fields.

Story Viewer

Uses a rendered local browser session to try loading the same public story feed the TikTok profile page requests.

View Videos

Shows the public video count and attempts to open the signed public video feed when TikTok exposes it to the rendered browser session.

Url Tracker

Paste a TikTok URL, resolve the handle, and jump straight into the profile lookup.

User Tracker

Favorites and recent lookups act as your local account tracker inside the browser.

Full Analysis

Summarizes the public account state, availability of extra signals, and the current lookup source.

Email2Username Tool

Shown in the HTML as requested, but intentionally disabled in this public-data build because it would rely on non-public account linkage.

Reposts Viewer

Uses a rendered local browser session to show public repost entries and their available public video streams when TikTok exposes them.

Download Avatar

Downloads the current account's public profile image through the local server.

FILE MODE FIX

The CORS error came from opening the page with `file://`.

The script now falls back to http://127.0.0.1:3000 when opened from disk, and the server now sends CORS headers for the API routes.

PUBLIC DATA ONLY

Unavailable data stays unavailable.

Some public TikTok pages expose handle, user ID, bio, language, stats, image, and timing fields. Stories, reposts, follower lists, and account-type signals are only shown when the public payload actually contains them.

PRIVACY

Privacy notes for this private app.

Browser behavior

The frontend sends the typed handle to the same server so the server can fetch the public TikTok page and return structured profile data.

No built-in analytics

This build does not include analytics, tracking pixels, or contact-form storage by default.

Network dependency

Live lookups require the server to reach TikTok. If outbound network access is blocked, the lookup endpoint will fail until access is restored.

Minimal private app with no payment or storefront data.
